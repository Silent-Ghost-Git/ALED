## What is about
<b>ALED</b> (Ankur's local examination dashboard (terrible name ik) ) is an examination dashboard for students to use for their exams preperations. This dashboard
allows students organize their worksheets, make plans and make tasks accordingly to maximize their workflow. They have also accsess
to resource like a timer for settings goals and chatGPT link to a random ahh convo for motivation. It is modular and allows great flexibilty for
all kinds of exam patterns and portions.
